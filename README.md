# is.gd_ChromeExtension

## 概要
URL短縮サービスの老舗である、イギリスのis.gdを簡単に使うための拡張機能です。

GoogleGeminiを仕様して作成しました。
https://gemini.google.com/share/ec94093f3ac4

## 使用方法
1. ダウンロードしたzipを解凍し、任意のディレクトリに格納してください。
2. Chromeから `chrome://extensions/` を開く。
3. 右上の「デベロッパーモード」をONにする。
4. `パッケージ化されていない拡張機能を読み込む` から、1のディレクトリを選択。
5. ピン留めをすると使いやすいです。
